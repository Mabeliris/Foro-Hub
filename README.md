## 1. Descripción del proyecto

Este proyecto forma parte del programa **ONE (Oracle Next Education)** en la formación de **Java con Spring Boot**.  
Consiste en una **API REST** desarrollada con **Spring Boot**, que implementa un **CRUD completo** para la gestión 
de recursos (crear, leer, actualizar y eliminar).  

El objetivo del desafío es demostrar habilidades en el desarrollo de **servicios REST**, manejo de **entidades JPA**
y buenas prácticas en arquitectura y diseño de aplicaciones backend.

## 4. Tecnologías utilizadas

- **Java** – Lenguaje de programación principal.  
- **Spring Boot** – Framework para construir servicios REST .  
- **MySQL** – Base de datos relacional para persistencia de datos.  
- **Spring Data JPA (Hibernate)** – Abstracción de la capa de persistencia para trabajar con entidades y repositorios de manera eficiente.  
- **Flyway** – Herramienta para gestionar migraciones y versionamiento de la base de datos, asegurando consistencia entre entornos.
